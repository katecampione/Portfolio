Projects

Yelp Camp

A multi-user full-stack Yelp clone built with Node, Express, Mongo, Mongoose, Passport.js, and Bootstrap. This app features secured user signup and login functions and allows logged-inusers to post, edit and delelte campgrounds.

Patatap Clone

This is a clone of a popular web app called Patatap. Users click on a keyboard key and the app responds with a sound and colorful animation sprites. The Patatap clone was built using HTML5/CSS, while Paper.js was used to animate circles with unique features for each key, and Howler.js was implemented to trigger a specific audio file for each key.
Caution: this game is highly addictive.
Note: This web app only functions on devices with a physical keyboard.

Color Game

While this game is not nearly as fun as Patatap, it's a great way to master rgb color codes. This game was built using HTML5, CSS, and Vanilla Javascript.

MUSEUM OF CANDY

Museum of Candy is a responsive website with Voodoo Donut-like vibes using photos from Unsplash. It is built utilizing HTML5, CSS, Bootstrap, and Flexbox.
